This project is a visually engaging coffee shop landing page built with HTML, CSS, and JavaScript. It features a bold hero section with a warm color gradient, eye-catching product imagery, and strong promotional text (“Make Your Day Great With Our Special Coffee”) to capture user attention.
The navigation bar includes key sections such as Home, About, Menu, Testimonial, Gallery, and Contact, providing a clear structure for exploring the site. Interactive elements, powered by JavaScript, enhance user experience by enabling dynamic behavior and responsiveness to user actions.
The design focuses on aesthetics and branding, using rich colors and typography to create a cozy, inviting coffee-themed interface. While the current version is not mobile responsive, it demonstrates foundational frontend development skills, including layout design, styling, and basic scripting for interactivity.
